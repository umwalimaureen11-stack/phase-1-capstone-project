About This Project

Book Explorer is a responsive web application built using HTML, Tailwind CSS, and JavaScript.

It allows users to search books using the Open Library API and save their favorite books using localStorage.

Features

  • Browse and search books from Open Library API
  • Add books to favorites
  • Persistent storage using localStorage
  • Fully responsive design with Tailwind CSS
  • Mobile-first approach
